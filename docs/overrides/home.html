Jailrun

A radically simple and coherent way to run your stack. Powered by FreeBSD.

Get started Learn more
# define your stack
jail "web" {
  setup {
    nginx { url = "hub://nginx/rolling"; }
  }
  forward {
    http { host = 8080; jail = 80; }
  }
}

# bring it up
$ jrun up web.ucl

# done. accessible at localhost:8080
